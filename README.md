# 📊 Instagram Fake Account Detection - EDA Project

## 📌 Overview
Project ini bertujuan untuk melakukan **Exploratory Data Analysis (EDA)** terhadap dataset profil Instagram guna memahami pola dan karakteristik akun fake.

EDA difokuskan pada:
- Distribusi fitur
- Perbandingan akun fake vs real
- Feature engineering sederhana
- Insight behavior akun fake

---

## 📂 Dataset
Dataset diambil dari Kaggle:

👉 https://www.kaggle.com/code/durgeshrao9993/fake-instagram-profile-detection-model/input

Dataset ini berisi fitur-fitur profil Instagram seperti:
- Profile picture
- Username pattern
- Fullname characteristics
- Followers & following
- Account activity
- Label fake (0 = real, 1 = fake)

Dataset ini umum digunakan untuk:
- klasifikasi akun fake
- analisis perilaku akun sosial media :contentReference[oaicite:0]{index=0}
